## Overview
This project is an AI-based Vision System designed to detect mind wandering and attention levels in real time using Computer Vision and Deep Learning techniques.

The system analyzes facial expressions, eye movement, and visual attention patterns through webcam input to identify whether a person is focused or distracted.

This project can be useful in:
- Smart Education Systems
- Online Learning Monitoring
- Driver Attention Monitoring
- Productivity Analysis
- Human Behavior Research

---

## Features
- Real-time Webcam Monitoring
- Face Detection
- Eye Movement Tracking
- Attention State Prediction
- Deep Learning-based Classification
- Real-Time Output Display
- Accuracy Visualization
